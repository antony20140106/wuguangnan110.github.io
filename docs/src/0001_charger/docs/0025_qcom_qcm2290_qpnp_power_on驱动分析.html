概述

高通qcm2290平台，目前有专门的开关机原因检测驱动，分析一下。

参考

  • android L 关机流程图

根据上图得知：

  • 关机时会调用msm_restart_prepare和do_msm_poweroff接口控制pmic关机。

  • 开机时qpnp-power-on.c驱动会打印开机原因。
