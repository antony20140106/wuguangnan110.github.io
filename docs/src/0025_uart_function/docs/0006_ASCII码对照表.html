概述

计算机串口通信都是以二进制的0、1来进行通信的，我们所说的发送字符即也是ASCII码、数值。其发送是转换成二进制，而二进制一般是从十六进制转换来的，所以，这两都需转换为十六进制，然后变为二进制进行传输，通过串口发送所对应的十六进制数就相当于发送了ASCII码。而在接收端，则是把这些接收到的二进制进行解析，对应ASCII码。

ASCII码 是现今最通用的单字节编码系统，并等同于国际标准ISO/IEC 646。
