### C#中string类使用Remove方法来移除指定位置的字符

```
举例，字符串string strA="ABCDEFG";

（1）移除从第4个字符开始后面的所有字符串可使用下列语句

string result=strA.Remove(3);//索引从0开始，第一个字符代表0

string result=strA.Remove(3，3);//索引从0开始，第一个字符代表0
```
