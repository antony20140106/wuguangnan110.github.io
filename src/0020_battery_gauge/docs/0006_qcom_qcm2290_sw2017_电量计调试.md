# 电池bms基本参数确认

* 容量 2800mah
* 标称电压：7.6V
* CV 8.7
* CC 
  * 0~10度 0.2C
  * 10~20度 0.5C
  * 20~45度 0.7C
* 标准放电电流：0.2C (-20~60℃)
* 持续放电电流:4A (-10~60℃)
* 最大放电电流:6A (-20~60℃)
* OVP： 4.425±0.020V  软件锂保最低在4.425-0.020 = 4.405V保护 再根据CV精度 ±0.4%，充电IC最高可能电压是4.3674V，双节x2最终计算8.7348v
* 过充电解除电压：4.225±0.035V
* 过放电检测电压：2.500±0.05V
* 过放电解除电压：2.800-2.900-3.000V 主选
* 过电流保护电流：6.0-10.0A
* 充电过电流保护电流：3.5-10.0A
